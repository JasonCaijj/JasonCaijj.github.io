随机变量及其分布

一、随机变量及其分布

1. 随机变量的定义
  • 定义1: 在随机试验$E$中，$\Omega$是相应的样本空间，如果对$\Omega$中的每一个样本点$w$，有唯一一个实数$X(w)$与之对应，那么就把这个定义域为$\Omega$的单值实值函数$X=X(w)$称为随机变量
  • 离散型随机变量：取值为有限或可列加个值
  • 非离散型变量（连续型离散变量）：取值为一个区间，有无限个值
2. 随机变量的分布函数

  • 定义2: 设$X$是一个随机变量，对于任意实数$x$，称函数$F(x)=P(X\leq x), -\infty < x < +\infty$为随机变量$X$的分布函数

  • 分布函数$F(x)$性质:

    • 对于任意实数$x$，有
      • $0\leq F(x) \leq 1$
      • $\lim\limits_{x\to -\infty}F(x)=0$
      • $\lim\limits_{x \to + \infty}F(x)=1$
    • $F(x)$单调不减，即当$x_1\leq x_2$时，有$F(x_1)\leq F(x_2)$
    • $F(x)$是$x$的右连续函数，即$\lim\limits_{x \to x_0^+}F(x)=F(x_0)$
3. 离散型随机变量及其分布律
  • 定义3：若一维离散型随机变量$X$的取值为$x_1$，$x_2$，$…$，$x_n$，称相应的概率为离散型随机变量$X$的分布律或分布列或概率函数
  • 性质：
    • 非负性：$p_i \geq0, i=1, 2, …$
    • 规范性：$\sum\limits_{i=1}^\infty p_i=1$
4. 连续型随机变量及其分布律
  • 概率密度函数(连续) = 分布律（离散）
  • 定义4: 设$E$为随机试验，$\Omega$是相应的样本空间，$X$是$\Omega$上的随机变量，$F(x)$是$X$的分布函数，若存在非负函数$f(x)$使得则称$X$为一维连续型随机变量，$f(x)$称为$X$的概率密度函数
  • 性质：
    • 非负性：$f(x) \geq0, - \infty < x < +\infty$
    • 规范性：$\int_{-\infty}^{+\infty}f(x) \,dx=1$

二、常用的离散型随机变量

1. 二项分布
  • 伯努利试验: 随机试验$E$，只关心某一事件$A$发生还是不发生，即该随机试验只有两种可能的试验结果：$A$和$\bar{A}$
  • $n$次独立重复的伯努利试验叫做n重伯努利试验
  • 在$n$重伯努利试验中$A$事件发生了$k$次，即$\{X=k\}$的概率为 称随机变量$X$服从参数$n$，$p$的二项分布，记为$X \sim B(n,p)$
  • 常见场景：
    • 相互独立抛骰子$n$次，出现$k$点的次数
2. 泊松分布
  • 设随机变量$X$的取值为0，1，2，…，$n$，…，相应的分布律为称随机变量$X$服从参数为$\lambda$的泊松分布，记为$X \sim P(\lambda)$
  • 与计数过程相关
  • 性质：
    • 作为二项分布的一种近似。在二项分布计算中，当$n$较大时，计算结果非常不理想，如果$p$较小，而$np=\lambda$适中时，经常用泊松分布的概率值近似取代二项分布的概率值

3. 超几何分布
  • 设有$N$件产品，其中有$M(M\leq N)$件是不合格品。若从中不放回地抽取$n(n \leq N)$件，设其中含有不合格品的件数为$X$，则$X$的分布律为称$X$服从参数为$N$、$M$和$n$的超几何分布，记为$X \sim H(N, M, n)$，其中$N$、$M$和$n$均为正整数。
4. 几何分布与负二项分布
4.1 几何分布
  • 在伯努利试验中，记每次试验中$A$事件发生的概率$P(A)=p (0<p<1)$，设随机变量$X$表示$A$事件首次出现时已经试验的次数，则$X$的取值为1，2，…，$n$，…，相应的分布律为则称随机变量$X$服从参数$p$的几何分布，记为$X\sim Ge(p)$
  • 性质：
    • $P(X>m+n | X>m)=P(X>n)$
    • 几何分布具有无记忆性的性质，即条件概率值只与$n$有关，与$m$无关
4.2 负二项分布
  • 在伯努利试验中，记每次试验中$A$事件发生的概率$P(A)=p (0<p<1)$，设随机变量$X$表示$A$事件第$r$次出现时已经试验的次数，则$X$的取值为1，2，…，$n$，…，相应的分布律为则称随机变量$X$服从参数$r$，$p$的负二项分布，记为$X\sim NB(r, p)$，当$r=1$时，即为几何分布。
